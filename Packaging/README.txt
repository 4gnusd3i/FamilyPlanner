FamilyPlanner v0.1.0
====================

Start:
  Double-click Start-FamilyPlanner.cmd

The launcher starts FamilyPlanner on:
  http://localhost:5080

Default data location:
  %LocalAppData%\FamilyPlanner

Language shortcuts:
  Start-FamilyPlanner.cmd nb
  Start-FamilyPlanner.cmd en

Advanced options:
  Start-FamilyPlanner.cmd -NoBrowser
  Start-FamilyPlanner.cmd -Language no-NB
  Start-FamilyPlanner.cmd -Language en-US
  Start-FamilyPlanner.cmd -Portable

Portable mode stores data in a data folder beside this launcher. The default
mode is safer for upgrades because your data survives replacing this app
folder with a newer release.

This v0.1.0 package is unsigned. Windows SmartScreen may show a warning until
FamilyPlanner has a code-signing certificate and signed release pipeline.
