FamilyPlanner v0.1.0
====================

Start:
  Double-click FamilyPlanner.exe

FamilyPlanner starts on:
  http://localhost:5080

Default data location:
  %LocalAppData%\FamilyPlanner

The app opens the browser automatically after startup. If the browser does not
open, go to http://localhost:5080 manually.

Your data is not stored in this extracted app folder. It stays in LocalAppData
so it survives replacing this folder with a newer release.

This v0.1.0 package is unsigned. Windows SmartScreen may show a warning until
FamilyPlanner has a code-signing certificate and signed release pipeline.
